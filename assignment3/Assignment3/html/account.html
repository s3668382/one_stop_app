Minimun Fearure: Account Creation

User Story:

As a soccer fan, I want to create an account in the app, so I can customise my favourite player and have personalised information in my account.

Design Notes and considerations:

User can have control of their account and manage information about themselves.

Validation Criteria:

Task Create an account
If succeeds the user will have an account to help store information on their favourite players among other things
If failed no data will be kept between uses of the app meaning that no favourites can be saved and the user will have to search for things they want to see each time they use the app
