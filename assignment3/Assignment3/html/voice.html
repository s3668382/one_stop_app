Minimun Fearure: Voice Assistance

User Story:

As a soccer fan, I want an application that I can search or ask anything about soccer by voice without using keyboard. This function will be very convenient for search infomation. Users no longer have to be afraid of typing the wrong letters or not knowing how to write the names of teams, players, coaches, etc, especially, old people they who have trouble about visual when they type something.

Design Notes and considerations:

To open Voice Assistance, user taps on the search bar on the top and then choose micro symbol on the right hand side

A voice assistant is a digital assistant using voice recognition. This allows the users to have assistance while using the application able to communicate to the application asking for particular players, teams and leagues and following out those instructions.

Validation Criteria

Task Search with a voice input
If succeeds after saying a phrase the phone will input it into the search
If failed the voice input will not work and no input will be registered
