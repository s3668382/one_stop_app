Minimun Fearure: Burger Menu

User Story:

As a mobile user i find that a lot of apps dont have an easily accessible menu from any where in the app. Having an easy way to navigate through an application would be amazing. something in the form of a burger menu where all relevent sections of the app would appear and i am able to simply click it and move on to the next section.

Design Notes and considerations:

When deciding what to put in the burger menu we felt best to leave the vr to scpecifically the player card as it could cause coding issues down the track and we watned to keep it as simple as possible for the user.

Validation Criteria:

Task Open the Burger menu in the top left of the screen
If succeeds The user will be able to view the open menu and interact with it
If failed If the menu fails to open then the page will remain the same and the user will view what ever page they were currently on
